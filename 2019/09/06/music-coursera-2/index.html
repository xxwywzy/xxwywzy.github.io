调和调号（Keys and Key Signatures）

调号（Key Signatures）

  • 如果将升半音和降半音符号全部写在乐谱中，会使得乐谱很混乱，不易读
    • 下图展示了 B 大调下的一段乐谱

  • 对于同一调式下的音，我们可以通过调号来减少升降符号的使用
    • 下图为添加了 B 大调调号的同一段乐谱

  • 在 B 大调调号下，所有的 F、C、G、D、A 都升半音
    • 该乐谱没有任何额外的变音记号
  • 对于额外存在的变音记号，需要单独注明出来（仅对其后，一小节之内适用）
    • 还原符号♮用于表示该音符不进行任何升降（有时写出只是为了提请注意）

五度圈（Circle of 5ths）

  • 我们可以使用五度圈来快速地找出调号
  • 五度圈是从 C 开始依次加五度构成的音（纯五度）
    • 顺时针的方向是上行五度，逆时针是下行五度

  • 当升降号出现在调式中时，其总是满足特定的顺序的
    • 对于升半音符号，顺序永远为 F C G D A E B
    • 对于降半音符号，顺序永远为 B E A D G C F
  • 我们可以通过五度圈快速识别调号的名称，或是根据名称写出该调号
    • 如果调号是使用升半音符号的，我们从五度圈的 C 开始顺时针数即可
    • 如果调号是使用降半音符号的，我们从五度圈的 C 开始逆时针数即可
  • 注意：没有调号是同时包含升降符号的
  • 下面的图片给出了常见谱号下调号的位置（不同谱号下中央 C 的位置有所不同）
    • 注意大调与小调存在对应关系

小调（Minor Keys）

  • 如果我们要找到一个大调所对应的小调，只需要在该大调的音阶中找出其六度音，作为小调的主音
    • 以 A 大调为例，其对应的小调为 F# 小调： 1=A - 2=B - 3 = C# - 4=D - 5=E - 6 = F#
  • 同理，如果我们要找到一个小调对应的大调，只要在该小调的音阶中找出其三度音，作为大调的主音
    • 从音程角度来说，即选择小调主音的小三度音作为大调主音即可（包含三个半音）
    • 以 A 小调为例，其三度音为 C，所以对应的大调为 C 大调

小调的类型

  • 小调有很多种类型，这里仅介绍下面三种：

自然小调（Natural Minor）

  • 这是最简单的小调类型，只需要取大调的六度音作为主音即可
  • 下图给出了 A 大调对应的 F# 自然小调

和声小调（Harmonic Minor）

  • 和声小调是在自然小调的基础上将七度音升半音得到
  • 对于 F# 小调来说，E 为其七度音，将 E 升半音即可得到 F# 和声小调

旋律小调（Melodic Minor）

  • 旋律小调的上行和下行有所不同
    • 对于上行，将六度音和七度音升半音
    • 对于下行，还原六度音和七度音（即和自然小调一致）

音程（Intervals）

  • 为了计算并更好地描述音程，我们需要两个方面的信息：
    • 音程的数值
    • 音程的质量

音程的数值

  • 即两个音直接包含了几个音，直接从低音向高音数即可（包含低音与高音）

音程的质量

  • 对于同样数值的音程，其包含半音数量的不同导致了其质量的不同
  • 常用的确定音程质量的方法如下：
    1. 首先我们选择以低音为主音的大调，观察高音是否位于该大调中
    2. 如果位于大调中，则其音程质量如下表所示：
    音程数值 音程质量
    同音 纯（Perfect）
    二度（2nd） 大（Major）
    三度（3rd） 大（Major）
    四度（4th） 纯（Perfect）
    五度（5th） 纯（Perfect）
    六度（6th） 大（Major）
    七度（7th） 大（Major）
    八度（8ve） 纯（Perfect）
    1. 如果不位于大调中，则根据其与大调中的音的关系（以半音为间隔）确认音程质量，具体如下所示：
      • 减（Diminished） → 小（Minor） → 大（Major） → 增（Augment）
      • 减（Diminished） → 纯（Perfect） → 增（Augment）

实例

  • 低音为 C，高音为 A，因此观察 A 是否位于 C 大调中
  • A 位于 C 大调中，音程数值为 6，因此为大六度

  • 低音为 C，高音为 Ab，因此观察 Ab 是否位于 C 大调中
  • A 位于 C 大调中，Ab 为其减半音，音程数值为 6，因此为小六度

  • 低音为 D#，高因为 C#，而 D# 大调并不存在，我们有以下两种处理方法：
    1. 取消所有的变音记号
      • 想象我们判断的是 D-C，容易得出这是一个小七度
      • 如果 D-C 是一个小七度，那么 D#-C#，也是一个小七度
    2. 取消低音的变音记号：
      • 想象我们判断的是 D-C#，容易得出这是一个大七度
      • 现在我们将 D 换回 D#，相当于在大七度的基础上减少了一个半音，因此该音程是一个小七度

复合音程（Compound Intervals）

  • 对于数值超过八度的音程，为了表达更加简洁，我们可以将其与低于八度的音程关联起来
  • 下图中，与该音程关联的为纯四度，因此我们称之为复合纯四度（compound perfect 4th）

附加线和谱号（Ledger Lines and Clefs）

谱号（Clefs）

  • 谱号用于表示不同音高的五线谱
  • 我们可以以中央 C 的位置来识别不同的谱号

高音谱号（Treble clef）

  • 高音谱号也被称为 G 谱号
    • 其符号中圆圈对准的第二行表示音符 G

低音谱号（Bass clef）

  • 低音谱号也被称为 F 谱号
    • 其符号中两点对准的第四行表示音符 F

中音谱号（Alto clef）

  • 中音谱号也被称为 C 谱号
    • 其符号中心对准的第三行表示中央 C

次中音谱号（Tenor clef）

  • 次中音谱号也被称为 C 谱号（同上）
    • 只是符号中心对准第四行

  • 我们可以通过中央 C 将两个五线谱连接起来

附加线（Ledger Lines）

  • 我们可以通过在五线谱上方或下方添加附加线来延长五线谱
