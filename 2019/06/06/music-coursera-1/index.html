音符（Musical Notes)

八度音（Octave）

  • 八度音是指来自不同音域的相同音名，下图展示了五线谱上相邻的八度音：

  • 可以看到如果一个音名在线上，则其相邻的八度音一定在两条线之间（反之亦然）
    • 我们将较低的那个 C 称为中央C（middle C），它一般位于键盘的中央

音程（Intervals）

  • 音程指两个音的高低关系

全音和半音（Tones and semitones）

  • 两个音之间的最小距离称为半音
    • 两个半音距离构成一个全音
  • 如果只考虑白键，我们有如下两组半音
    • B 和 C
    • E 和 F

C 大调（The C major scale）

  • 在了解了全音和半音之间的关系之后，我们就可以推导出 C 大调

  • C 大调是一个于 C 音开始的音乐的大调
    • C 音为主音（tonic）
    • 这是一个自然音阶组成的大调（没有升降号）
      • 其特点是从一个八度音到其相邻八度音，其中包含 2 个半音和 5 个全音

理解三和弦的内部结构

首先我们计算三和弦中首尾两个音之间的音程：

  • 对于 C 大三和弦：
    • 其音程包含了 7 个半音，被称为纯五度（perfect fifth）

  • 对于 A 小三和弦：
    • 其音程同样为纯五度

接着我们计算首音和中间音的音程：

  • 对于 C 大三和弦：
    • 其音程包含了 4 个半音，被称为大三度（major third）

  • 对于 A 小三和弦：
    • 其音程包含了 3 个半音，被称为小三度（minor third）

  • 基于上述结果我们可以得出三和弦的命名规则：
    • 大三度 + 小三度（纯五度）：大三和弦
    • 小三度 + 大三度（纯五度）：小三和弦
    • 小三度 + 小三度（减五度）：减三和弦

三个主要和弦

  • 在音阶中，不同的音有不同的名称：
    • 第一个音称为主音（Tonic）
    • 第二个音称为上主音（Supertonic）
    • 第三个音称为中音（Mediant）
    • 第四个音称为下属音（Subdominant）
    • 第五个音称为属音（Dominant）
    • 第六个音称为下中音（Submediant）
    • 第七个音称为下主音（Subtonic）
  • 对于一个音阶，以下三个和弦最为重要：
    • 主和弦：以主音开头的和弦
    • 下属和弦：以下属音开头的和弦
    • 属和弦：以属音开头的和弦

  • 音阶中不同的音符适合与这三个和弦中不同的和弦搭配
    • 搭配的和弦中应包括该音符
